### USE CASES ###

#### Run script by URL ####
```
source <(curl -s http://some.domain/path/to/script.sh)
```

```

source <(curl -s https://raw.githubusercontent.com/train-to-cupertino/droplet-tools/main/scripts/profit_server/common/install-docker-compose-new.sh)
```
